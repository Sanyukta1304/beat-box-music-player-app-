# 🎵 Music Streaming Web App

A modern, fully responsive music streaming web application built using **React + Vite**.  
This project simulates a real-world music platform experience with interactive playback controls, playlist management, authentication, and persistent user data — all powered entirely on the front-end.

---

## 🚀 Live Features

### 🎧 Advanced Music Player
- Play / Pause functionality  
- Next / Previous controls  
- Shuffle & Repeat modes  
- Real-time progress bar with seek control  
- Volume adjustment slider  
- Live "Now Playing" section  

### 📂 Song Management
- Landing page with featured songs  
- Full song library (grid layout)  
- Album details page  
- Add / Remove songs from playlist  
- Favorites (Like songs) feature  
- Search by title, artist, album, or genre  

### 👤 User System
- Login & Register (localStorage simulation)  
- Protected routes (authentication required)  
- Personalized User Dashboard  
- Persistent playlists and favorites  

### 🎨 UI & User Experience
- Dark / Light mode toggle  
- Responsive navbar  
- Smooth transitions and animations  
- Toast notifications for actions  
- Fully responsive (Desktop + Tablet + Mobile)  

---

## 🛠 Tech Stack

- **React.js**
- **Vite**
- **React Router DOM**
- **Context API (Global State Management)**
- **CSS3**
- **localStorage (Data Persistence)**
